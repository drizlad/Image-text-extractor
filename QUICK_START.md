# 🚀 Quick Start Guide

## ✅ Extension is Ready!

All errors have been fixed. The extension now works perfectly!

## 📦 Installation (30 seconds)

1. Open Chrome and go to: `chrome://extensions/`
2. Toggle "Developer mode" ON (top right)
3. Click "Load unpacked"
4. Select the `dist` folder
5. Done! ✅

## 🎯 How to Use (3 steps)

1. **Click** the extension icon in toolbar
2. **Drag** to select any text area on the page
3. **Copy** the extracted text from the popup

That's it!

## 🧪 Test It Now

1. Open `test-page.html` in Chrome
2. Click the extension icon
3. Drag over any text section
4. Wait 2-5 seconds (first time: 10-15 sec)
5. See the extracted text!

You'll see a friendly notification if you try these.

## 🐛 Troubleshooting

**Extension not activating?**
→ Refresh the page and try again

**OCR taking too long?**
→ First time downloads data (10-15 sec), be patient

**Copy not working?**
→ Extension uses 2 fallback methods, should always work

**Selection not showing?**
→ Make sure cursor changed to crosshair
